Efimov
- Description: Solve Efimov Hamilitonian by QTT.
- Recommand environment: Julia v1.11
- Files:
- functions.jl includes most of the functions used in this project.
- efimov_e^x_setup.jl is the setup of the Efimov Hamiltonian, need to create folder efimov_e^x, info.txt, and mps.h5 first, the later two are in efimov_e^x.
- muti_dmrg.jl is the main file.
